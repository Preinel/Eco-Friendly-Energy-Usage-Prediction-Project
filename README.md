# Eco-Friendly-Energy-Usage-Prediction-Project
Created by Preinel Abreu(PRN: 23030124211) , Joel Matthew (PRN: 23030124234) under the guidance of Vikas Kumar Choudhary Sir at SICSR
This project predicts household energy consumption patterns
to recommend eco-friendly practices.

#PREREQUISITES
1) Make sure you have:
  Python 3.8 or higher
  and Git
3) Open Cmd and run: pip install pandas numpy seaborn matplotlib scikit-learn
  

#HOW TO RUN THE FILE
1) Open Git Bash and run: git clone https://github.com/Preinel/Eco-Friendly-Energy-Usage-Prediction-Project.git
2) Run: cd Eco-Friendly-Energy-Usage-Prediction-Project
3) Run: python project.py
